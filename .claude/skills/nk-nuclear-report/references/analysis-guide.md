# 분석 가이드

## 입력
- `sources/YYYY-MM-DD/index.json` — 전체 태깅 현황 파악용
- `sources/YYYY-MM-DD/items/src-XXX.json` — `new`/`update` 태그 항목만 개별 열람
- `reports/YYYY/MM/*.md` — 이전 7일 보고서

## 분석 과정

### 1. 신규 소스 분석 (tag: new)
- **중요도 평가:** 높음 / 중간 / 낮음
  - 높음: 핵 프로그램 직접 진전, 군사적 위협 증대, 주요 외교 변화
  - 중간: 기존 동향의 새로운 분석, 전문가 평가
  - 낮음: 배경 정보, 간접 연관
- **카테고리 분류:** 핵실험 / 미사일 / 우라늄농축 / 외교·제재 / 군사력 / 사이버 / 기타
- **이전 보고서 연관:** 직접 연관 없음 / 맥락적 연결 있음

### 2. 기존 보도 추적 (tag: update)
- **원본 보고서:** 어떤 날짜의 어떤 항목과 연결되는지
- **변경/추가 정보:** 구체적으로 무엇이 새로운지
- **추적 가치:** 높음 (유의미한 진전) / 낮음 (부수적 업데이트)

### 3. 제외 소스 (tag: reported)
- **제외 근거:** 원본 참조 + 새 정보 없음 확인

### 4. 주제별 흐름 분석
최근 7일 동향과 오늘 새 정보를 주제별로 정리:
- 핵시설 동향
- 미사일 개발 동향
- 외교/제재 동향
- 기타 주요 동향

## 산출물 1: analysis.md

```markdown
# 연관관계 분석 (YYYY-MM-DD)

## 분석 요약
- 신규 소스: N건, 업데이트: N건, 제외: N건

## 신규 소스 분석
### [src-001] 뉴스 제목
- **중요도:** 높음
- **카테고리:** 외교·제재
- **분석:** 왜 중요한지 1-2문장

## 기존 보도 추적
### [src-010] 뉴스 제목
- **원본:** YYYY-MM-DD 보고서 "항목 제목"
- **변경사항:** 구체적 새 정보
- **추적 가치:** 높음

## 제외 소스
### [src-015] 뉴스 제목
- **제외 근거:** 04-05 항목과 동일, 새 정보 없음

## 주제별 흐름 분석
(주제별 7일 동향 + 오늘 새 정보)
```

| 소스 ID | 제목 | 제외 근거 |
|---------|------|----------|

## 보고서 구성 방향
- 요약 방향: (강조할 내용)
